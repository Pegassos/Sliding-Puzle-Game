# Sliding-Puzle-Game
Sliding puzzle game where the player should move the grids to make them in ascending order from 1 - 15 to win.

# Controls
Arrow keys,
Mouse click
or Buttons

# Functionalities
Number of moves counter 
and reset game button

# How to run 
Download files and run the following commands:

### `npm i express`
### `node index.js`
